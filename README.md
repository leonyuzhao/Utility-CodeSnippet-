Utility [CodeSnippet]
====================

Utility classes used for quick implementation of standard operations.

Setup
====================

Add relevant files into project folder.

* **C#**

`using Utility;`

* **VB.Net**

`Imports Utility`

Structure
====================

* **Utility.IO.File**
* **Utility.Web.IMail**
* **Utility.Web.Gmail**
* **Utility.Web.IContent**
* **Utility.Web.HttpContent**
